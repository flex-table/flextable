# FlexTable

The public home for **FlexTable** — a desktop database manager for PostgreSQL, MySQL, SQLite, MongoDB, and Redis.

> The application source is private. This repository is for **downloads**, **release notes**, and **issue / feedback** tracking.

## Download

Get the latest build from the [**Releases**](../../releases) page (macOS · Windows · Linux). Once installed, the app keeps itself up to date automatically.

## Report a bug / request a feature

- 🐞 **Bug?** → [Open a bug report](../../issues/new?template=bug_report.yml)
- 💡 **Idea?** → [Open a feature request](../../issues/new?template=feature_request.yml)
- 💬 **Question / general feedback?** → [Discussions](../../discussions)

When reporting a bug, please include your **FlexTable version**, **OS**, and the **database type** involved — it makes fixes much faster.
