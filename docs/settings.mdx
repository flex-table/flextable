Open **Settings** to configure the app.

![Settings](/images/10-settings/settings.png)

## What you can configure

- **Appearance** - theme (light or dark) and display preferences.
- **AI Assistant** - connect a provider with your own API key and base URL. See [AI Assistant](/ai-assistant).
- **Editor and grid** - query-editor and data-grid preferences.
- **Transaction mode** - when on (the default), your pending table edits commit together as one transaction; when off, each change autocommits. A footer badge shows the current mode.
- **Keyboard shortcuts** - override any default. See [Keyboard shortcuts](/keyboard-shortcuts).

<Tip>**Auto-updates.** FlexTable checks for updates and lets you install them with a click. New releases are included for your license's update window.</Tip>
